Vframe (id) Vframe类

源码： vframe.js

Properties

  • id {String} vframe id
  • owner {VOM} VOM对象
  • view {View} view对象
  • viewUsable {Boolean} view是否可用，即view的interact事件有没有派发

Static Properties

Methods

  • childrenAlter () 通知子vframe有变化
  • childrenCreated () 通知所有的子view创建完成
  • fire (name, data, remove, lastToFirst) 触发事件
  • locationChanged (loc, chged) 通知当前vframe，地址栏发生变化
  • locationUpdated (loc) 通知location更新
  • mountVframe (id, viewPath, viewInitParams, byHand) 加载vframe
  • mountView (viewPath, viewInitParams) 加载对应的view
  • mountZoneVframes (HTMLElement|String, viewInitParams, autoMount) 加载当前view下面的子view，因为view的持有对象是vframe，所以是加载vframes
  • newViewCreated () 转场动画时或当view启用刷新动画时，新的view创建完成时调用
  • oldViewDestroy () 转场动画时或当view启用刷新动画时，旧的view销毁时调用
  • on (name, fn, insertOrRemove) 绑定事件
  • prepareNextView () 转场动画时或当view启用刷新动画时，为新view准备好填充的容器
  • Vframe. root (vom) <static> 获取根vframe
  • un (name, fn) 解除事件绑定
  • unmountVframe (id, useAnim) 销毁vframe
  • unmountView (useAnim, isOutermostView) 销毁对应的view
  • unmountZoneVframes (HTMLElement|String, useAnim) 销毁某个区域下面的所有子vframes
  • useAnimUpdate () 是否启用场景转场动画，相关的动画并未在该类中实现，如需动画，需要mxext/vfanim扩展来实现，设计为方法而不是属性可方便针对某些vframe使用动画

Events

  • alter (e) 子孙view修改时触发
  • created (e) 子孙view创建完成时触发
  • viewInteract (e) view可交互时触发
  • viewUnmount () view卸载时触发
