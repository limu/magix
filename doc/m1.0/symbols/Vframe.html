Vframe (id) Vframe类

源码： vframe.js

Properties

  • id {String} vframe id
  • owner {VOM} VOM对象
  • view {View} view对象
  • viewInited {Boolean} view是否完成初始化，即view的inited事件有没有派发

Static Properties

Methods

  • fire (name, data, remove, lastToFirst) 触发事件
  • mountVframe (id, viewPath, viewInitParams, byHand) 加载vframe
  • mountView (viewPath, viewInitParams) 加载对应的view
  • mountZoneVframes (HTMLElement|String, viewInitParams, autoMount) 加载当前view下面的子view，因为view的持有对象是vframe，所以是加载vframes
  • on (name, fn, insertOrRemove) 绑定事件
  • un (name, fn) 解除事件绑定
  • unmountVframe (id) 销毁vframe
  • unmountView () 销毁对应的view
  • unmountZoneVframes (zoneId) 销毁某个区域下面的所有子vframes

Events

  • alter (e) 子孙view修改时触发
  • created (e) 子孙view创建完成时触发
  • viewInited (e) view初始化完成后触发
  • viewUnmounted () view卸载时触发
